# Stuck in the Mud (Java Game)

A Java Swing implementation of the classic dice game "Stuck in the Mud".

## Rules
- Roll 5 dice
- Dice showing 2 or 5 become "stuck"
- Other values add to your score
- Game ends when all dice are stuck

## Technologies
- Java
- Swing (GUI)
- Event-driven programming
